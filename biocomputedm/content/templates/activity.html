Upload Data

Upload a new dataset
  • Allows you to upload data to Biocompute-DM's file system
  • Files must be either .zip or .gz
  • Files will be placed in a quarantined zone and unzipped.
  • Unzipped file structures should constitute be as follows /unzipped folder name/data only (i.e. no nested directories)
  • Uploaded data can be added to Biocompute-DM's management framework by choosing the submit data option

Submit Data

Submit a new dataset
  • Submit a dataset from your uploaded files or from your group's directory
  • Submitted data must be directly below its containing folder (e.g. /upload1/data_file_1)
  • Submitted data (from your upload directory) becomes visible to your group once added
  • Submitted data can then be deconvoluted using available plugins to generate per sample information

Add an Investigation

Add a new investigation.
  • Create a parent investigation that can hold multiple samples and sample groupings

Add a Sequencing Submission

Add a new sequencing submission.
  • Add your sequencing run to the system to begin the deconvolution process.

Add a Flow Cytometry Submission

Add a new flow cytometry submission.
  • Add your flow cytometry run to the system to begin the deconvolution process.
