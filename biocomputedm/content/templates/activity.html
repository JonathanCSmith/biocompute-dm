Biocompute-DM Activity Window

Here you can find a list of activities available for you to perform.

In the toolbar above you can find summaries for your activities.

In the toolbar below you can find site, and user, specific information.

{# #}

View Available Pipelines

View Pipelines
  • Use the link above to view the pipelines available to you as well as any specific requirements they have for data. All pipelines come with a documentation booklet that may be of interest to you.

Upload Data

Upload a new dataset
  • Allows you to upload zipped data to a personal storage area within Biocompute-DM. Uploaded zips then become available to send to pipelines.

Select Data to Submit to a Pipeline

Select Data for Submission!
  • Select the data (from your uploads) that you wish to submit to a pipeline. Please note, some pipelines expect files in a very specific format, so please refer to the pipelines of interest before submitting!

Run a Pipeline on a Data Group

Run a Pipeline!
  • Select a data group from those already available (these are produced by pipelines) in order to run a new pipeline on it. In future you will be able to create custom data groups.

Create a Custom Sample Group

Add a New Sample Group
  • Use this option to create a custom grouping of samples, samples are created by pipelines (not manually). Please note that all of the samples grouped together must have come from the same pipeline type

Create a Project

Add a New Project
  • Use this option to create a new project. Projects can be used to group samples, sample groups and documents.
