Upload Data

Upload a new dataset
  • Allows you to upload data to Biocompute-DM's file system
  • Files must be either .zip or .gz
  • Files will be placed in a quarantined zone.
  • Unzipped file structures should contain no directories (i.e. after unzipping, only data files are present).
  • Uploaded data can be added to Biocompute-DM's management framework by choosing the submit data option.
  • The data are unzipped when you add them to a submission.

Submit Data

Submit a new dataset
  • Submit a dataset from your uploaded files or from your group's directory
  • Submitted data must be directly below its containing folder (e.g. /upload1/data_file_1)
  • Submitted data (from your upload directory) becomes visible to your group once added
  • Submitted data can then be deconvoluted using available plugins to generate per sample information

Create a Sample Group

Add a New Sample Group
  • Use this option to create a custom grouping of samples.
  • Please note that all of the samples grouped together must have come from the same pipeline type

Create a Project

Add a New Project
  • Use this option to create a new project.
  • Projects can be used to group samples, sample groups and documents.
